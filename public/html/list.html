How to represent list in html ?

in html, there are three kinds of lists which you choose depending on your purpose.

  Three kind lists: 1 unordered lists 2 ordered lists 3. Describled lists

Unordered Html Lists

unordered html lists means that items in a list just list its value and comes along a icro before every item

First, Let me show a example.

        <ul>
            <li>Car</li>
            <li>Food</li>
            <li>Computer</li>
            <li>Book</li>
        </ul>

below html code will show:

  • Car
  • Food
  • Computer
  • Book

There is a icon before every item, this is default style, also you can alter it by list-style-type attribute in <ul> tag.
  there are four style for it, look at following table.

Style Description
list-style-type:disc The list items will be marked with bullets(default)
list-style-type:circle The list items will be marked with circles
list-style-type:square The list items will be marked with square
list-style-type:none The list items will be marked with none

Ok, Let us try some change.
Using Following html code fragement:

        <ul style="list-style-type:square">
            <li>Car</li>
            <li>Food</li>
            <li>Computer</li>
            <li>Book</li>
        </ul>

Its effect will look like this:

  • Car
  • Food
  • Computer
  • Book

Ordered Html Lists

ordered html lists means that items in a list list its value and comes along a number before every item

First, Let me show a example.

        <ol>
            <li>Car</li>
            <li>Food</li>
            <li>Computer</li>
            <li>Book</li>
        </ol>

below html code will show:

  1. Car
  2. Food
  3. Computer
  4. Book

There is a icon before every item, this is default style, also you can alter it by list-style-type attribute in <ul> tag.
  there are four style for it, look at following table.

tyle Description
type="1" The list items will be marked with numbers(default)
type="A" The list items will be marked with uppercase letters
type="a" The list items will be marked with lowercase letters
type:"I" The list items will be marked with uppercase roman numbers
type:"i" The list items will be marked with lowercase roman numbers

Ok, Let us try some change.
Using Following html code fragement:

        <ol type="A">
            <li>Car</li>
            <li>Food</li>
            <li>Computer</li>
            <li>Book</li>
        </ol>

Its effect will look like this:

  1. Car
  2. Food
  3. Computer
  4. Book

Last, We talk about describled lists

to use it ,we can use <dl> , <dd>and <dl> tags to achieve,For example: 

        <dl>
            <dt> Coffee </dt>
            <dd> Black and White </dd>

            <dt> Milk </dt>
            <dd> White cold drink </dd>
        </dl>
It will look like :
Coffee

Milk
