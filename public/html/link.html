How to use link tag rightly in html page?

1 Link to absolute address

if you want to link to be absolute address, you should put your address into href value.

Example: <a href="http://www.baidu.com">baidu.com</a>

Result: baidu.com

2 Link to reletive address

Use reletive address to link resources on the same website

Example: <a href="index.html">index</a>

Result: index






















3 Link to a bookmark

To link a bookmark ,you should define a bookmark which just implement by id attribute firstly

For example: <p id="top"Top2> Top </p>

To use a link to refer that bookmark ,you should refer #top in href attribute
