How to use table tag rightly in html page?

Table is a important tag in html tag which can represent data in humanable way.

So, In this page, I will show you how to use table tag to create table basicly.

I will show you what parts can be in one table

First, We define a simple table, using following html code fragement, Code:

<table>
    <tr>
        <td>Name</td>
        <td>Age</td>
        <td>Address</td>
    </tr>

    <tr>
        <td>Jiangjian</td>
        <td>23</td>
        <td>Shanghai</td>
    </tr>
</table>

<table> represent a table
<tr> represent a table row
<td> represent a table data
<th> represent a table head

Flowing code will be showed like this:

Jiangjian 23 Shanghai
